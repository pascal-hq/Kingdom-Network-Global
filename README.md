## ✨ Features

- ✅ **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- ✅ **Glassmorphism UI** - Modern frosted glass effect throughout
- ✅ **Image Slider** - Auto-rotating hero section with manual navigation
- ✅ **Daily Bible Verse** - Fetches from Bible API with local fallback
- ✅ **Department Pages** - 5 dedicated pages with detailed information
- ✅ **Prayer Request Form** - Submit prayer needs directly
- ✅ **Newsletter Signup** - Collect email subscribers
- ✅ **Smooth Scrolling** - Animated navigation between sections
- ✅ **Scroll Reveal Animations** - Elements fade in as you scroll
- ✅ **Active Nav Highlighting** - Current section highlighted in menu
- ✅ **Mobile Hamburger Menu** - Optimized for mobile devices
- ✅ **SEO Friendly** - Meta tags and semantic HTML structure

## 🛠 Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, flexbox, grid, animations
- **JavaScript (ES6+)** - Vanilla JS, no frameworks
- **Font Awesome 6** - Icon library
- **Google Fonts** - Playfair Display & Inter fonts
- **Formspree** - Form handling (email integration)
- **Bible API** - Daily verse fetching
